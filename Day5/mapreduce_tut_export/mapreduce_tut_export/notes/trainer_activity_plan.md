# Trainer Activity Plan - MapReduce

## Use Case
SmartRide wants to count words/status/city names from trip logs using Hadoop MapReduce.

## Activities
1. Explain why big files are split into HDFS blocks.
2. Show how input splits are created for MapReduce.
3. Explain RecordReader line-by-line conversion into key-value records.
4. Explain Mapper output as intermediate key-value pairs.
5. Explain Shuffle and Sort.
6. Explain Reducer aggregation.
7. Explain Driver Program configuration.
8. Run built-in WordCount.
9. Build and run custom Java WordCount.
10. Use ResourceManager UI to observe the application.

## Tools
- Hadoop HDFS
- Hadoop MapReduce
- YARN ResourceManager UI
- Java JDK
- Maven
- Linux terminal
