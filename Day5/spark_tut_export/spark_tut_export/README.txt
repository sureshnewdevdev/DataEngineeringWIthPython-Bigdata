SmartRide Spark Tutorial - Sample Files

Files included:
1. index.html - Complete TUT-style classroom tutorial.
2. data/trips.csv - Sample SmartRide trip data.
3. code/smartride_spark_intro.py - PySpark DataFrame example.
4. code/smartride_spark_sql.py - Spark SQL example.

How to run sample code:
1. Install Java and PySpark.
2. From this folder, run:
   spark-submit code/smartride_spark_intro.py
   spark-submit code/smartride_spark_sql.py

Purpose:
These files help explain Spark basics, Spark SQL, Spark architecture, Driver, Executor, Task, SparkSession, and the Spark ecosystem using one simple ride-hailing use case.
