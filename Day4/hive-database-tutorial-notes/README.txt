Apache Hive Database Tutorial Notes
===================================

Files:
- hive-database-tutorial-notes.html

How to use:
1. Open the HTML file in Chrome, Edge, or Firefox.
2. Use the left navigation to move between the 13 sections.
3. Tick completed topics; progress is saved locally in your browser.
4. Use the Practice MCQ and Formal Timed Exam.
5. Double-click section headings to add them to your notes box.

Topic coverage:
- Hive database concepts
- Hive warehouse directory
- Hive Metastore
- CREATE/USE/DESCRIBE/ALTER/DROP DATABASE
- Managed vs external tables
- Loading data
- Partitions, bucketing, performance
- Retail analytics classroom use case
- Interview questions and answers
